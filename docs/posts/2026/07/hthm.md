黑糖话梅是一款基于多款开源字体融合开发的中文圆体字体，整体风格圆润可爱，最初于 2019 年通过微博 @孤鹜先森 及微信公众号 「霞鹜」 发布，主要面向手机字体美化及日常设计场景。随后项目正式迁移至 GitHub 开源维护，并采用 SIL Open Font License 1.1（OFL 1.1） 协议发布，允许个人及企业在遵守协议的前提下免费使用。

![黑糖话梅](https://wkphoto.bj.bcebos.com/8718367adab44aedf5cdafbfa31c8701a08bfbee.jpg)

从字形表现来看，黑糖话梅是一款偏圆润、可爱、亲和的中文字体。它的整体骨架接近圆体和粗黑体之间的结合，字形较为饱满，笔画厚度较高，但通过圆角化处理减少了传统黑体的硬朗感，使文字看起来更加柔和。字体的横竖笔画相对均衡，转折处多采用圆润处理，整体视觉压力较低，适合用于轻松、活泼、生活化的设计场景。

黑糖话梅最明显的特点是笔画圆润且带有一定弹性。中文部分的字面较大，内部空间相对紧凑，能够在标题场景中形成较强的识别度。部分笔画的收笔带有轻微弧度和软化效果，使文字不显得生硬，同时保留了较好的结构稳定性。相比完全手写风格字体，它的字形更加规整；相比普通圆体，它又多了一些可爱和趣味感。
